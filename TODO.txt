
Задание требует создать небольшое React-приложение, которое будет позволять пользователям добавлять, просматривать и удалять элементы из списка дел.

Общие требования:
1. Настройка проекта:
   - Создать простой проект на React (без сложной структуры папок).
   - Использовать функциональные компоненты и хуки React (например, useState, useEffect).

2. Основные функции:
   - Добавление задачи: Позволить пользователю добавлять новую задачу в список с помощью текстового поля и кнопки "Добавить".
   - Просмотр списка: Отображать список задач. Каждая задача должна содержать текст и кнопку "Удалить".
   - Удаление задачи: Позволить пользователю удалить задачу из списка.
   - Хранение данных на сервере: Данные должны храниться на бэкенде (можно использовать базу данных в памяти или SQLite). Бэкенд нужно реализовать с использованием Nest.js.

3. UI/UX:
   - Простая стилизация для организации списка и поля ввода.
   - Показывать сообщение, когда в списке нет задач, например, "Нет задач для отображения".

4. Бонус (по желанию, если хватит времени):
   - Добавить флажок для каждой задачи, чтобы пометить её как выполненную.
   - Использовать localStorage для сохранения списка задач между перезагрузками страницы.
